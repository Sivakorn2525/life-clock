# นาฬิกาชีวิต (Life Clock)

แอปนาฬิกาชีวิตแบบอะนาล็อก-ดิจิทัล ติดตั้งลงมือถือ/คอมพิวเตอร์ได้เหมือนแอปจริง (PWA)
ไม่ต้องมีเซิร์ฟเวอร์ ไม่มีฐานข้อมูล ข้อมูลทั้งหมดเก็บในเครื่องผู้ใช้ (localStorage) เท่านั้น

## โครงสร้างไฟล์
```
life-clock/
├── index.html          หน้าแอปหลัก
├── style.css           ดีไซน์ (โทนหมึกเข้ม-ทองเหลือง-กระดาษไข)
├── app.js              ตรรกะทั้งหมด: คำนวณอายุ, วาดหน้าปัด, จัดการเป้าหมาย
├── manifest.json        ให้เบราว์เซอร์รู้จักว่าเป็นแอปที่ติดตั้งได้
├── service-worker.js    แคชไฟล์ไว้ใช้งานออฟไลน์
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## วิธีติดตั้งให้ใช้งานได้จริง (ต้องมี URL ที่เปิดผ่าน HTTPS)

PWA จะ "ติดตั้งลงเครื่อง" ได้ก็ต่อเมื่อเปิดผ่านที่อยู่เว็บที่เป็น HTTPS (หรือ localhost)
เลือกวิธีใดวิธีหนึ่งด้านล่าง ฟรีทั้งหมด ไม่ต้องเขียนโค้ดเพิ่ม:

### วิธีที่ 1: GitHub Pages (แนะนำ ถ้ามีบัญชี GitHub อยู่แล้ว)
1. สร้าง repository ใหม่ เช่น `life-clock`
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไป (ลาก-วางผ่านหน้าเว็บ GitHub ก็ได้)
3. ไปที่ Settings → Pages → เลือก branch `main` → Save
4. รอ 1-2 นาที จะได้ลิงก์ เช่น `https://ชื่อบัญชี.github.io/life-clock/`
5. เปิดลิงก์นั้นบนมือถือ/คอมพิวเตอร์ แล้วกด "เพิ่มลงหน้าจอโฮม" (มือถือ) หรือไอคอน "ติดตั้ง" ที่แถบ URL (Chrome/Edge บนคอมพิวเตอร์)

### วิธีที่ 2: Netlify Drop (ไม่ต้องสมัครบัญชีก็ได้)
1. เข้า https://app.netlify.com/drop
2. ลากทั้งโฟลเดอร์ `life-clock` วางลงในหน้าเว็บ
3. จะได้ลิงก์ทันที เปิดลิงก์แล้วติดตั้งได้เลย

### วิธีที่ 3: เปิดจากเครื่องตัวเอง (ทดสอบเร็วๆ ก่อน)
เปิด index.html ตรงๆ ผ่าน `file://` จะใช้งานหน้าปัดและเป้าหมายได้ปกติ
แต่เบราว์เซอร์จะไม่เสนอให้ "ติดตั้งเป็นแอป" และ service worker จะไม่ทำงาน (ต้องมี http/https)
ถ้าอยากทดสอบแบบเสมือนจริงในเครื่อง ให้รันเซิร์ฟเวอร์เล็กๆ เช่น
```
cd life-clock
python3 -m http.server 8080
```
แล้วเปิด `http://localhost:8080` ในเบราว์เซอร์ (localhost ถือว่าใช้ได้เหมือน HTTPS)

## การติดตั้งบนแต่ละแพลตฟอร์ม
- **Android (Chrome):** เปิดลิงก์ → เมนู ⋮ → "ติดตั้งแอป" หรือแบนเนอร์ที่ขึ้นมาเอง
- **iPhone/iPad (Safari):** เปิดลิงก์ → ปุ่มแชร์ → "เพิ่มลงหน้าจอโฮม"
- **Windows/Mac (Chrome/Edge):** เปิดลิงก์ → ไอคอนจอคอมพิวเตอร์เล็กๆ ท้ายแถบ URL → "ติดตั้ง"

## การปรับแต่ง
- โทนสีทั้งหมดอยู่ใน `style.css` ส่วน `:root` (ตัวแปร `--brass`, `--sage`, `--parchment` ฯลฯ)
- ค่าอายุขัยเริ่มต้น (80 ปี) แก้ไขได้ในแอปเองที่แท็บ "ตั้งค่า" ไม่ต้องแก้โค้ด
- ถ้าต้องการเปลี่ยนไอคอน แก้ `gen_icons.py` (สคริปต์ต้นทางของไอคอน) หรือแทนที่ไฟล์ใน `icons/` ตรงๆ (ต้องเป็นขนาด 192x192 และ 512x512)

## ความเป็นส่วนตัว
ไม่มีการส่งข้อมูลวันเกิดหรือเป้าหมายออกจากเครื่องผู้ใช้เลย ทุกอย่างเก็บด้วย `localStorage`
ของเบราว์เซอร์เท่านั้น หากล้างข้อมูลเบราว์เซอร์หรือเปลี่ยนเครื่อง ข้อมูลจะหายไป (ยังไม่มีระบบ sync ข้ามเครื่อง)
